🔽 **A GitHub Pages–hosted web tool for downloading selected Chromium builds in ZIP format**


## 📌 Project Description

**Chromium Build ZIP Downloader** is a static web application hosted on **GitHub Pages** that provides a simple and user-friendly interface for downloading **Chromium** builds as ZIP archives.  
The tool eliminates the need to manually browse Chromium build repositories by offering an intuitive UI directly in the browser.

Because the project is built as a client-side application, it requires **no backend, no installation, and no configuration** — it runs entirely in the browser and is available instantly via GitHub Pages.

## ⚙️ Features

- ✅ Fully static application hosted on **GitHub Pages**
- ✅ No backend or server required
- ✅ Select Chromium release channel:
  - **Stable**
  - **Beta**
  - **Dev**
  - **Canary**
- ✅ Automatically fetch the latest build number for the selected channel
- ✅ Ability to manually set a custom build number
- ✅ Download Chromium as a **ZIP archive**
- ✅ Modern, responsive UI
- ✅ Multilingual interface with instant language switching

---

## 🖥️ How It Works

1. Open the application via GitHub Pages
2. Select a Chromium release channel
3. Verify or adjust the build number
4. Click **“Download Chromium (ZIP)”**
5. The ZIP file is downloaded directly from official Chromium sources

## ⚠️ Notes

- This project does **not** host Chromium binaries.
- All downloads are fetched from **official Chromium repositories**.
- The tool serves purely as a frontend helper for easier access to builds.
